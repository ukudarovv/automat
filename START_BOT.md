# Инструкция по запуску бота

## Способ 1: Через командную строку (рекомендуется)

Откройте новый терминал и выполните:

```bash
python run_bot.py
```

Бот запустится и будет показывать логи в реальном времени.

## Способ 2: Через batch файл

Дважды кликните на файл `start_bot.bat`

## Способ 3: Через PowerShell

```powershell
Start-Process python -ArgumentList "run_bot.py" -WindowStyle Normal
```

## Проверка работы бота

1. Откройте Telegram
2. Найдите бота: @atyrauhub_backend_bot
3. Отправьте команду `/start`
4. Бот должен ответить с меню выбора

## Остановка бота

Нажмите `Ctrl+C` в окне, где запущен бот.

## Логи

Логи сохраняются в файл `bot.log` в корне проекта.
